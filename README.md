# Mobile Computing Lab Practicals

This repository contains the code and resources for a series of practical exercises completed for a Mobile Computing Lab. The projects are built using **Java** and **Android Studio**, demonstrating fundamental concepts of Android application development.

## Practical List

1. **Practical 1:** Activity Lifecycle  
2. **Practical 2:** Text Display  
3. **Practical 3:** Toast  
4. **Practical 4:** Simple Calculator  
5. **Practical 5:** Intent  
6. **Practical 6:** UI Components  
7. **Practical 7:** Button OnClickListener  
8. **Practical 8:** Canvas  
9. **Practical 9:** Alert Dialog  
10. **Practical 10:** File Handling  
11. **Practical 11 & 12:** Connectivity  
12. **Practical 13:** CRUD with SQLite  
13. **Practical 14:** Animation  
14. **Practical 15:** Shared Preferences  
15. **Practical 16:** Image Download  
16. **Practical 17:** Video Player
